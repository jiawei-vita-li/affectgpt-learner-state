# Roadmap

## Current Version: Reference MVP

- Sample interview JSON
- Streamlit demo
- Open-vocabulary emotion timeline
- Evidence-grounded pain points
- Product opportunity cards
- Structured rationale
- Product and quality metrics
- Local upload mode for text, JSON, image, and video demo inputs
- Hugging Face facial expression classification for uploaded images/videos
- Basic tests

## V0.2: Better Research Workflow

- Add human review state: accepted, edited, rejected
- Add Markdown export for research summaries
- Add filters by participant segment, product area, and journey stage
- Add a bad-case gallery
- Improve uploaded-file parsing and add privacy reminders before analysis

## V0.3: Multi-Interview Synthesis

- Aggregate pain points across multiple interviews
- Add frequency and severity matrix
- Add opportunity prioritization view
- Add contradiction and caveat tracking

## V0.4: Evaluation Harness

- Add evidence-grounding rubric
- Add actionability rubric
- Add emotion consistency checks
- Add regression tests for sample insight outputs

## V0.5: Privacy and Governance

- Add redaction workflow for sensitive quotes
- Add consent status handling
- Add blocked-use warnings
- Add audit log export

## Long-Term Direction

- Optional local transcript upload
- Local-only model adapters for ASR, OCR, OpenFace / MediaPipe-style landmarks, and audio feature extraction
- Research repository integrations
- Collaboration with annotation/review workflows
